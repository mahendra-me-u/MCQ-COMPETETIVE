🔧 Firebase Setup Required

Follow these steps to connect your Firebase database:

1. Go to Firebase Console

2. Create a new project (or use existing)

3. Click "Add app" → Choose "Web" () icon

4. Register your app and copy the config object

5. Enable Firestore Database (in Build menu)

6. Set Firestore Rules to test mode:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
